## 🎨 그림판 (paintjs)

바닐라 자바스크립트로 제작된 그림판입니다.  
붓의 크기와 색을 바꿀 수 있으며 배경을 칠할 수 도 있습니다.  
그림을 그린 후 이미지로 저장할 수 있습니다!  
  
Painting Board made with VanillaJS  
you can draw on web-canvas and save it!
